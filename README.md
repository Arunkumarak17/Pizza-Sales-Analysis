**Pizza Sales Analysis – SQL & Power BI Project**

**Project Overview**

This project analyzes pizza sales data to generate meaningful business insights using SQL and Power BI. The goal is to evaluate sales performance, identify top-selling products, and understand customer ordering patterns.

SQL was used for data cleaning, transformation, and KPI calculations. The processed data was then visualized in Power BI through an interactive dashboard.

**Objectives**

1. Analyze total revenue and total orders

2. Identify top and bottom performing pizzas

3. Evaluate sales by category and size

4. Analyze daily and monthly sales trends

5. Determine peak order hours

**Tools & Technologies**

1. SQL Server (Data extraction & analysis)

2. Power BI (Dashboard & visualization)

3. Excel (Data preprocessing)

**Key KPIs**

1. Total Revenue

2. Total Orders

3. Average Order Value

4. Total Pizzas Sold

5. Sales by Category

6. Sales by Size

**Dashboard Features**

1. Revenue and order trend analysis

2. Best and worst-selling pizzas

3. Category-wise and size-wise performance

4. Time-based sales insights

**Skills Demonstrated**

1. SQL querying (JOIN, GROUP BY, Aggregate Functions)

2. Data Cleaning & Transformation

3. Data Modeling

4. DAX Calculations

5. Business Intelligence Reporting
